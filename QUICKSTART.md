## API Examples

### Create a task
```bash
curl -X POST http://localhost:8000/api/v1/tasks \
  -H "Content-Type: application/json" \
  -d '{"name": "My Task", "notes": "Task description"}'
```

### Get tasks in a project
```bash
curl http://localhost:8000/api/v1/projects/{project_gid}/tasks
```

### Create a section
```bash
curl -X POST http://localhost:8000/api/v1/projects/{project_gid}/sections \
  -H "Content-Type: application/json" \
  -d '{"name": "To Do", "project": "project_gid"}'
```
